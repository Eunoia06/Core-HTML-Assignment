HTML/CSS
Basic building blocks for web development

The primary scripting language for front-end development

CSS Preprocessors

Tools and frameworks like Bootstrap or Flexbox/Grid for responsive layouts

To run high-performance code written in languages like C, C++, or Rust in the browser
